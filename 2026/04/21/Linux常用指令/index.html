Linux常用指令

Author: DizzyGreen Date: April 21, 2026  18:42:41 Category: Linux操作系统

Linux常用指令

目录操作

  1. ls

    即 list 列出目录下所有项

    例：

    -l 参数 以详细模式查看

    -a 参数 all 显示所有文件，包括隐藏文件

  2. pwd

    即 print working directory 显示当前工作目录

  3. cd

    即 change directory 切换目录

    • ~ 当前用户主目录
    • . 当前目录
    • .. 上一级目录

  4. mkdir

    即 make directory 创建目录

    -p参数 将路径的层次目录全部创建

  5. rmdir

    即 remove directory 删除空目录

    如果目录为空则删除失败

  6. rm

    remove 删除文件或目录

    -rf r表示recursive，f表示force

  7. cp

    copy 复制文件或目录

  8. mv

    move 复制文件或目录

归档压缩

  1. 归档

    tar 即 tape archive 档案打包

    c， create 创建档案

    v，verbose 显示详情

    f，file

  2. 解包

    -C 参数 指定目标目录，默认当前目录

    x， extract

  3. 归档并压缩

软链接

类似快捷方式

ln link

-s 表示 soft 软链接（默认为硬链接）

添加用户

sudo 表示以管理员身份执行

-m 表示在/home 下添加用户目录

修改用户密码

删除用户

超级用户

  • 首次使用需要给root设置密码

  • 切换到root用户

    su， switch user

  • 退出

文件权限

  1. chmod

    change file mode 修改文件访问权限

    o 表示 other 别人 ，a all 所有人，u user 自己，默认修改自己和本组权限

    +w 表示增加写 write 权限，-w 表示删除写权限

  2. chown

    change owner 修改文件属主

    -R 表示递归

环境变量

  • 定义环境变量

  • 显示环境变量

  • 查看所有环境变量

    printenv
