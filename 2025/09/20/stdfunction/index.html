std::function

什么是 std::function？

std::function 是一个类模板，它可以存储、复制和调用任何可调用对象，只要该对象的调用签名与 std::function 的模板参数匹配。

可调用对象包括：

  • 普通函数
  • 函数指针
  • Lambda 表达式
  • 仿函数（重载了 operator() 的类对象）
  • 类的成员函数
  • 被 std::bind 绑定的表达式

基本语法

示例：

  • std::function<void()>：一个无参数、无返回值的函数包装器。
  • std::function<int(int, int)>：一个接受两个 int 参数并返回一个 int 的函数包装器。
  • std::function<double(std::string)>：一个接受 string 参数并返回 double 的函数包装器。
