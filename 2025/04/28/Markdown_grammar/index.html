Author: DizzyGreen Date: April 28, 2025  16:30:11 Category: markdown

markdown 的基本语法

文本

  • 文本书写

    直接写入内容即可。

  • 换行

    有些编辑器需要在行尾加两个空格，有些可以直接换行。

  • 加粗和斜体

    • 在文本的左右两侧添加两个*生成加粗文本。
    • 在文本的左右两侧添加一个*生成斜体文本。
    • 在文本的左右两侧添加三个*生成斜体加粗文本。
    • 所有’*’都可以用_代替，但更推荐使用*。

  • 删除线

    在文本左右两侧添加两个~使文本带上删除线，可以和加粗和斜体混合使用。
